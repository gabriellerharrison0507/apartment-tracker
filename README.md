# RoomList

A personal move-in tracker for furnishing and organizing a new apartment.

Built for Gabby's move to Denver, CO in June 2026.

## Features

- Track items across 6 rooms: Kitchen, Bathroom, Bedroom, Living Room, Balcony, and Cleaning
- Mark items as **purchased** or **already own**
- Add prices, links, and notes to each item
- Compare multiple product options per item
- See estimated cost ranges and actual spend by room
- Progress tracking per room
- All data saved locally in your browser (no account needed)
- Reset to defaults at any time

## Usage

Open the live site: [gabriellerharrison0507.github.io/apartment-tracker](https://gabriellerharrison0507.github.io/apartment-tracker/)

Or clone and open `index.html` locally — no build step required.

## Tech

Single-file HTML/CSS/JS app. No frameworks, no dependencies, no backend.
Data is persisted via `localStorage`.
